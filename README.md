
# Arvis 프로젝트 - To-Do & 2개월 달력

업무 인수인계 및 일정 관리를 위해 제작된 **단일 HTML 기반 To-Do 관리 프로그램**입니다.

---

## 1. 프로그램 개요

Arvis는 육군 업무 환경을 고려하여 개발된 **다중 스토리지 기반 업무 관리 도구**입니다.

사용자는 **소속 + 직책**을 기준으로 개별 스토리지를 생성하여 관리할 수 있으며,
대과제와 소과제를 등록·수정·삭제할 수 있습니다.

등록된 일정은 **2개월 달력**에 자동 표시되며, 특정 날짜를 중요일(빨간날)로 지정하여 함께 관리할 수 있습니다.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 스토리지 관리 | 소속/직책별 스토리지 자동 생성 및 전환 |
| 과제 관리 | 대과제·소과제 등록, 수정, 삭제, 완료 처리 |
| 담당자/메모 | 담당자 및 메모 입력 |
| 일정 관리 | 시작일·종료일 지정 |
| 필터 | 전체 / 완료 / 진행중 / 기한초과 |
| 정렬 | 최신순, 오래된순, 기한순, 제목순, 담당자순 |
| 달력 | 현재월 + 다음월 표시, 일정 및 중요일 표시 |
| 백업 | JSON 내보내기 / 불러오기 |
| 자동 저장 | LocalStorage 기반 자동 저장 |

---

---

## 4. 사용 방법

### 프로그램 실행

1. `worklist_handoff_Assistant.html`을 브라우저에서 실행합니다.
2. 최초 실행 시 **소속**과 **직책**을 입력합니다.

### 과제 등록

- 대과제 선택 또는 생성
- 소과제 입력(선택)
- 담당자 / 메모 입력
- 시작일 / 종료일 지정
- 완료 여부 선택
- **과제 추가** 클릭

### 과제 수정 / 삭제

- **수정** 버튼으로 기존 과제 편집
- **삭제** 버튼으로 선택 과제 삭제
- 대과제 삭제 시 하위 소과제도 함께 삭제

### 필터 및 정렬

필터와 정렬 드롭다운을 이용해 원하는 방식으로 목록을 조회합니다.

### 달력 기능

- 일정 자동 표시
- 날짜 더블클릭 → 중요일(빨간 표시) 지정
- 중요일은 스토리지에 저장됩니다.

### 백업 / 복원

- **리스트 백업** → JSON 저장
- **리스트 불러오기** → JSON 복원

### 스토리지 전환

상단 **스토리지 선택**에서 다른 스토리지로 즉시 전환할 수 있습니다.

---

## 5. 주의 사항

- 인터넷 연결이 필요하지 않습니다.
- 데이터는 브라우저 LocalStorage에 저장됩니다.
- 민감한 군 정보는 입력하지 않는 것을 권장합니다.
